# House Price Prediction using Linear Regression

This project predicts house prices using the California Housing Dataset.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Project Workflow
1. Data Collection
2. Data Analysis
3. Data Preprocessing
4. Model Training
5. Model Evaluation

## Results
- MAE: 0.5332
- RMSE: 0.7454
- R² Score: 0.6065
